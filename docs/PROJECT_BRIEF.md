# CARLA Foundation Driving Demo

## Mission
Build a credible autonomous driving AI engineering demo using CARLA that demonstrates the full lifecycle of an end-to-end driving model: environment bootstrap, data collection, dataset engineering, model training, inference, evaluation, explainability, and deployment packaging.

## Operating Model
The human owner acts as Product Owner, Chief Architect, and Reviewer.
AI coding agents act as Senior Staff Engineer, ML Engineer, DevOps Engineer, and Test Engineer.

## Bootstrap Rule
Do not assume Python, Git, CARLA, CUDA, PyTorch, OpenCV, or environment variables are already installed.
The platform must discover, verify, and report missing dependencies with clear remediation steps.

## Non-Goals
This is not a production autonomous vehicle stack.
This is not a certified safety system.
This is not a Level 4 autonomous driving product.

## Phase 0 Goal
Create a reproducible AI engineering platform scaffold with diagnostics, documentation, config separation, and agent-ready task structure.
